# HSR Wordle

Customizeable (but not very much, really) wordle based on **Honkai: Star Rail** game by HoYoverse. All info is gathered either from [HSR wiki](https://honkai-star-rail.fandom.com/wiki/Honkai:_Star_Rail_Wiki) or from [Mar-7th/StarRailRes](https://github.com/Mar-7th/StarRailRes) repo.

It's available here: https://alyssurite.github.io/hsr-wordle/
